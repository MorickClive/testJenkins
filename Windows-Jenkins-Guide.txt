How to Install Windows Jenkins:
=========================
	- Get your installer
		- Go to https://www.jenkins.io/
		- Press download
		- Go to "Download Jenkins 2.222.4"
		- As good practice, run a virus scan on the downloaded file
		- Unzip/Run the .msi installer
	- Installer
		- Follow the install process, remember where you installed it ( we'll refer to it as [INSTALL PATH])
		- Follow the instructions Jenkins provides on your browser, remember the credential details! :D
	
	This can paste your install path here for reference if you wish:
	[INSTALL PATH] : 

=========================
How "Start" a Windows Jenkins Instance:
=========================
	- Find your install
		- Go to your Windows "start menu"(bottom left windows Icon) and type 'cmd', right click the "command prompt" application.
		- Select "Run as administrator" for Command Prompt.
		- Navigate in Command prompt to your[INSTALL PATH] 
			(Hint if you installed on another drive, instead of cd, make sure you enter the drive name e.g 'Z:' to access that drive)
	
	- Begin Jenkins
		- Run "jenkins.exe start"(hint: make sure you are in the correct folder to find the exe with the command line interface.)
		- Open your browser and navigate to http://localhost:8080/
		- Wait for the server to load the webpage!
		- You're done! :D

=========================
How to "Stop/Close" a Windows Jenkins Instance:
=========================
	Main:
		- Follow stages 1-3 on "How to start Windows Jenkins", replace "start" with "stop".

	Alternatively:
		Enter the following:
		http://localhost:8080/exit (click button to send POST request)
		
=========================
How to Uninstall Windows Jenkins:
=========================
	- Use your downloaded .msi installed
	- Select the remove option and confirm.
	- You've Uninstalled Jenkins!
